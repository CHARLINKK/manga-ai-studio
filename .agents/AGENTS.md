# Rules for Manga AI Studio Project

- **Commits:** Sempre que finalizarmos atualizações de código no projeto, entregue ao usuário, de forma proativa, os textos exatos que ele deve colar nos campos "Summary" e "Description" do GitHub Desktop para realizar o commit. Os textos devem ser profissionais e formatados em blocos de código.

- **Confirmação e Exclusão:** Sempre confirme com o usuário antes de tomar qualquer ação drástica, principalmente se for apagar algum arquivo ou pasta. Não tome atitudes que envolvam deleção ou mudanças destrutivas de arquitetura sem a autorização explícita do usuário.
