# Regras de Comportamento (Manga AI Studio)

- **Lembretes de Commit**: Sempre que o código for alterado, um novo executável for gerado, ou arquivos importantes (como `version.json`) forem modificados, o agente DEVE obrigatoriamente lembrar o usuário de realizar o **Commit e Push** no GitHub Desktop.
- **Textos de Commit**: Junto com o lembrete, o agente DEVE fornecer o texto exato (Summary e Description) formatado nos padrões do "Conventional Commits" com emojis, pronto para o usuário copiar e colar no GitHub Desktop.
